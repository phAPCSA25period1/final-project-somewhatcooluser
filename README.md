## 🔹 Roll Your Birthdate

## 🔹 What This Software Does
Roll Your Birthdate is a Java console-based game where players try to reach their birthdate by rolling dice. Players start with a random date. They must choose whether to add or subtract the year, month, or day to reach their birthdate. Two dice will be rolled, and the total of those rolls will affect what the player has chosen to do. Players must reach their birthdate in as few rolls as possible.

## 🔹 Who It’s For
This game is meant for anyone who is bored and wants something to do to pass the time.

## 🔹 How to Run the Program
Ensure Java JDK 8+ is installed. Compile all .java files. Run the program by running "java App".

## 🔹 Technical Overview
Uses:
- 4 classes: App, Date, Leaderboard, and RollDice.
- Array, ArrayList, and 2D Array.
- Scanner for user input.
